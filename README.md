# Alizah — Portfolio

Personal portfolio website built with Next.js and TypeScript, featuring a light/dark mode toggle, mouse-tracked ambient glow, scroll progress indicator, scramble text animation, and an interactive 3D scene on the home page.

## Pages

- **Home** — Stats, about, featured project, contact strip
- **Stack** — Full tech stack organized by category, each skill links to its official documentation
- **Experience** — Timeline of work history, research, and competitions
- **Projects** — Landing page with Tech and Creative subpages
- **Resume** — Resume highlights and PDF download
- **Contact** — Click to copy email, LinkedIn, GitHub, and resume download

## Built With

- **Framework** — Next.js 16 with App Router
- **Language** — TypeScript
- **Styling** — Tailwind CSS v4
- **3D** — React Three Fiber, Three.js, Drei
- **Icons** — React Icons
- **Deployment** — Vercel

## Features

- Light and dark mode
- Mouse-tracked ambient glow on every page
- Scroll progress bar
- Scramble text animation
- Interactive 3D scene
- Click-to-copy email
- Back to top button
- Fully responsive

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.
